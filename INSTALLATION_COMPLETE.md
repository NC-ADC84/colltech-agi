## 🎓 Usage Examples

### Example 1: Consciousness-Enhanced Processing

```python
from colltech_agi.src.agentic_mindsets_integration import (
    AgenticMindsetsIntegration,
    AgenticMode
)

agentic = AgenticMindsetsIntegration()

result = agentic.process_with_agentic_mindset(
    "What is the meaning of existence?",
    mode=AgenticMode.CONSCIOUS
)

# Access consciousness metadata
consciousness = result['agentic_metadata']['consciousness']
print(f"Meaning Score: {consciousness['meaning_score']:.2f}")
print(f"Existential Relevance: {consciousness['existential_relevance']:.2f}")
```

### Example 2: Stable Problem-Solving

```python
result = agentic.process_with_agentic_mindset(

```
